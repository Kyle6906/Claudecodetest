Lead Generation

API keys for Google Maps Lead Finder and Apollo.io Decision Maker search. Keys are stored locally and never leave your computer.

Google Maps API Key
Powers the Lead Finder — searches for businesses near your territory zip codes
How to get this key ▸ Debug shows the raw Google API response
Google Maps Platform (Places API + Geocoding API)
  1. Go to console.cloud.google.com and sign in with a Google account
  2. Create a new project (or select an existing one)
  3. Search for "Places API" and enable it, then enable "Geocoding API"
  4. Go to APIs & Services → Credentials and click "+ Create Credentials → API Key"
  5. Copy the key and paste it above — Google provides $200/month free credit (~40,000 searches)
  6. Optionally restrict the key to only Places API + Geocoding API for security
Apollo.io API Key
Find decision makers (Operations, Warehouse, Procurement managers) at any company
How to get this key ▸ Debug shows the raw Apollo API response
Apollo.io API Key (free tier available)
  1. Go to app.apollo.io and create a free account
  2. Once logged in, click your profile icon → Settings
  3. Go to Integrations → API
  4. Copy your API key and paste it above
  5. Free tier includes: people search (names & titles always visible), 50 email reveals/month
  6. Emails marked "needs reveal" require spending 1 export credit (50 free/month)
